jQuery Alert Panels
=============
-------------
仿照修改显示为bootstarp样式的弹窗

用法
-------------

```javascript
// 错误信息
$.alert("error", "这是错误信息");

// 成功信息
$.alert("warning", "这是警告信息");

//
// 自定义信息
$.alert({
				message: "这是普通信息",
				settings: {
					duration: 0
				}
			});
});
```
